# 🤖 SmartSync Intelligent Assistant

SmartSync Intelligent Assistant is a full-stack web-based chatbot system built using Node.js, Express.js, and MySQL. It simulates intelligent responses using a rule-based system and provides user authentication with chat history storage.

---

## 🚀 Features

- 🔐 User Authentication (Login & Registration)
- 💬 Chatbot Interface (Rule-based responses)
- 💾 Chat History Storage (MySQL Database)
- 🎤 Voice Input Support
- 🌙 Dark Mode UI
- ⚡ Fast and Responsive Design

---

## 🛠️ Technologies Used

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MySQL
- Other: Fetch API, LocalStorage
